# Example Dimensionality Reduction Data

Example data for dimensionality reduction plots (e.g., UMAP, t-SNE).
Contains cell embeddings, velocity, and clustering information.

## Usage

``` r
dim_example
```

## Format

A data frame with:

- Embeddings:

  First 4 columns: x, y coordinates and velocity

- clusters:

  Cell cluster assignments

- Additional metadata:

  Additional grouping variables
