# Build element_text from style spec

Converts a style specification to ggplot2::element_text()

## Usage

``` r
build_element_text(spec_path, base_size = 12, ...)
```

## Arguments

- spec_path:

  Character. Path to style specification

- base_size:

  Numeric. Base font size

- ...:

  Additional arguments to override spec

## Value

A ggplot2::element_text object
