# Create background layer for plots

Create background layer for plots

## Usage

``` r
create_bg_layer(
  data,
  x,
  palette,
  palcolor,
  alpha,
  keep_empty,
  facet_by,
  direction = "vertical"
)
```

## Arguments

- data:

  Data frame

- x:

  X column

- palette:

  Palette name

- palcolor:

  Custom colors

- alpha:

  Alpha value

- keep_empty:

  Keep empty levels

- facet_by:

  Faceting columns

- direction:

  Background direction

## Value

ggplot layer
