# Create base forest plot

Shared base plot creation for forest plots

## Usage

``` r
.create_forest_base(
  data,
  point_colors,
  point_size,
  point_border_width,
  point_border_size,
  line_colors,
  line_type,
  line_width,
  cutoff_vline_type,
  cutoff_vline_width,
  cutoff_vline_color,
  x_log10_scale
)
```
