# Combine multiple plots

Combine multiple plots

## Usage

``` r
combine_plots(
  plots,
  combine = TRUE,
  nrow = NULL,
  ncol = NULL,
  byrow = NULL,
  axes = NULL,
  axis_titles = NULL,
  guides = NULL,
  design = NULL
)
```

## Arguments

- plots:

  List of plot objects

- combine:

  Whether to combine

- nrow:

  Number of rows

- ncol:

  Number of columns

- byrow:

  Fill by row

- axes:

  Axis handling

- axis_titles:

  Axis title handling

- guides:

  Guide handling

- design:

  Custom design

## Value

Combined plot or list
