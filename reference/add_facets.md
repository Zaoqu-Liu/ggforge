# Add faceting to a plot

Add faceting to a plot

## Usage

``` r
add_facets(plot, facet_by, facet_scales, nrow, ncol, byrow, ...)
```

## Arguments

- plot:

  ggplot object

- facet_by:

  Column(s) to facet by

- facet_scales:

  Scale type for facets

- nrow:

  Number of rows

- ncol:

  Number of columns

- byrow:

  Fill by row

- ...:

  Additional arguments for facet functions

## Value

Faceted plot
